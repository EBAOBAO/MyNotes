# 基础

## 电学量

```tx
| 电学量 | 定义 | 单位 |
| :--: | :--: | :--: |
| 电荷 Q | 赋予基本粒子的一种属性 | C |
| 电流 I | 电荷移动的速率，$$I = \frac{dQ}{dt}$$ | A |
| 电压 U | 两端之间移动一个单位正电荷所需的能量 | V |
```

每个电流都具有一个数值和一个方向，开路能保证电流为 0 。

电压也具有大小和方向，且它只有在两点间才有意义。短路能够确保电压为 0 。
